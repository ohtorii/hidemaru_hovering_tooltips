# Visual Assist Xっぽい秀丸マクロ
Visual Assist X の Hovering Tooltips っぽい秀丸マクロを途中まで書いてみました。

関数とクラスのコメントをアウトプット枠へ表示するマクロです。
コメントはTooltipsで表示したかったんですが、秀丸エディタに機能が無いためアウトプット枠へ出力しました。
